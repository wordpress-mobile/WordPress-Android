使用 Android 裝置管理或建立 WordPress 網誌或網站：建立及編輯文章和網頁、上傳最愛的相片和影片、查看統計資料及回覆留言。

使用 Android 版 WordPress，你就能夠在掌上輕鬆發佈文章。窩在沙發上信手創作幾行俳句；趁著午休時間拍照並上傳；回覆最新的回應；或者查看統計資料，以瞭解今天的讀者來自哪些國家。

Android 版 WordPress 是開放原始碼專案，這表示你也可以貢獻己力協助開發此程式。深入瞭解：https://apps.wordpress.com/contribute/。

Android 版 WordPress 支援 WordPress.com 和執行 WordPress 4.0 或以上版本的自助託管 WordPress.org 網站。

需要應用程式的說明嗎？請造訪論壇：https://android.forums.wordpress.org/forum/troubleshooting 或使用 Twitter 推文給我們：@WPAndroid。
