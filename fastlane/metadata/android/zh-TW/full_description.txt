Android 版 WordPress 讓你隨時都能使用網頁發佈功能。 這是一套網站建立工具，還有許多功能等你來發掘！

建立
- 讓重大想法得以在網路上找到棲身之處。 Android 版 WordPress 是一套網站建立和網誌製作工具。 請善用這套工具建立你的網站。
- 從各種 WordPress 佈景主題中選擇最合適的外觀和樣式，然後使用相片、顏色和字型自訂你的網站，創造自己的獨特風格。
- 內建的「快速入門」提示可引導你完成設定基本功能，協助你成功設定新網站。 (我們不只是一套網站建立工具，也是你的合作夥伴，並適時給予加油喝采！)

統計資料
- 即時查看網站統計資料，追蹤網站上的活動。
- 探索每日、每週、每月和每年的洞察報告，追蹤哪些文章和頁面在一段時間內獲得最多流量。
- 使用流量圖查看訪客來自哪些國家/地區。

通知
- 取得留言、按讚和新關注者的通知，即時掌握訪客對網站的反應。
- 立即回覆新留言，這樣除了能夠保持雙向溝通，也能藉此讓讀者獲得回饋。

發表
- 使用編輯器建立更新、故事、相片隨筆和公告，任何內容都可以！
- 利用相機和相簿中的相片及視訊，或在免費的專業攝影應用程式內圖庫尋找適合的圖片，讓文章和頁面變得更加生動。
- 將構想儲存在草稿，等到有靈感時再繼續完成，或為新文章排定一個發表日期，這麼做可以讓網站保持更新，吸引訪客前來瀏覽。
- 新增標籤和類別，幫助新讀者探索你的文章，並觀察讀者數量成長的情況。

讀取器
- WordPress 不僅是一套網誌製作工具，你也可藉此與 WordPress Reader 中的作家社群交流。 透過標籤探索成千上萬的主題，探索新作者及機構，也可以關注你感興趣的對象。
-「儲存以便稍後使用」功能可以讓你保留感興趣的文章。

分享
- 設定自動分享，在發表新文章時通知社交媒體上的關注者。 自動轉貼到 Facebook 及 Twitter 等。
- 在文章中新增社交分享按鈕，訪客即可將內容分享到自己的社交網路，讓你的粉絲成為你的宣傳大使。

為什麼要選擇 WordPress？

市面上有很多網誌服務、網站建立器和社交網路， 為什麼要選擇 WordPress 建立你的網站？

WordPress 為超過三分之一的網頁提供技術支援， 在各領域之間廣為使用，例如有關個人嗜好的網誌、各種規模的企業、網路商店，甚至是網路上最大的新聞網站。 許多你喜歡的網站可能都選用 WordPress 運作。

使用 WordPress，你的內容歸你所有。  其他社交網路將你視為商品，並擅自佔有你發表內容的所有權。 但在 WordPress，你發表的所有內容皆為你的資產，甚至還能隨身帶著走。

WordPress 是一套開放原始碼網站建立工具，任何人都可以查看其原始碼，甚至貢獻一己之力。  其他服務和社交網路則是專有的封閉系統，你永遠無法確定他們的運作原理，也不知道他們正在執行哪些工作，有一天這些服務甚至還可能消失！

無論你是需要網站製作工具來建立網站，或只需要簡單的網誌製作工具，WordPress 都可以幫得上忙， 不只提供你精美的設計、強大的功能，還賦予你自由創作的空間。
