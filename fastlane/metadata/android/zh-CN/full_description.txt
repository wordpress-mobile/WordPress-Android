Android 版 WordPress 将 Web 发布权交到您的手里。它不仅是建站工具，还具有许多其他用途！

创建
- 给您的重要创意在网上找到安家之地。Android 版 WordPress 是网站生成器和博客创建器。使用它创建您的网站。
- 从丰富的 WordPress 主题中挑选合适的外观和风格，然后用照片、颜色和字体加以定制，打造出您独有的网站。
- 内置的快速开始技巧将指导您掌握基础的建站知识，帮助您建立成功的新网站。（我们不仅提供建站工具，还充当您的合作伙伴和啦啦队！）

统计数据
- 实时查看您网站的统计数据，随时了解您网站上的活动。
- 通过研究每日、每周、每月和每年的数据分析，跟踪哪些文章和页面在一段时间内获得最多流量。
- 使用流量图了解访客来自哪些国家或地区。

通知
- 获得有关评论、点赞和新粉丝的通知，从而实时了解人们对您的网站有何反应。
- 及时回复新出现的评论，使对话能持续进行和答谢读者。

发布
- 借助编辑器撰写最新动态、故事、照片随笔、公告等各种内容！
- 使用相机和相册中的照片和视频来生动呈现文章和页面，或者通过应用程序内置的专业摄影图片库（免费使用）找到最理想的图片。
- 将创意保存为草稿并在灵感再现时回头审视这些草稿，或者安排要在日后发布的新文章，让您的站点始终保持新鲜感和吸引力。
- 添加标签和类别以帮助新读者发现您的文章，同时观察读者群的增长情况。

读者
- WordPress 不仅仅是博客创建器，它还能让您在 WordPress 阅读器中与一群作者建立联系。根据标签来浏览数千个主题、发现新的作者和组织，以及关注激发您兴趣的人士。
- 使用“保存以备后用”功能留存吸引您的文章。

分享
- 设置自动分享功能，以便当您发布新文章时在社交媒体上告知您的粉丝。文章会自动交叉发布到 Facebook、Twitter 等社交媒体。
- 在文章中添加社交分享按钮，让访客可以与其关系网分享文章，并让您的粉丝成为您的传播大使。

为何使用 WordPress？

市面上有大量的博客服务、网站生成器和社交网络。为何要使用 WordPress 来创建网站？

有三分之一的网站都运行在 WordPress 上。业余博客、各种规模的企业、网店甚至互联网上最大的新闻网站都使用它。您喜爱的许多网站很可能都运行在 WordPress 上。

使用 WordPress 时，您自己的内容归您所有。其他社交网络将您视为商品，而且您发布的内容归它们所有。但如果使用 WordPress，您发布的任何内容都归您所有，而且去到哪里都能将这些内容带在身边。

WordPress 是一个开源建站工具，这表示任何人都能看到它是如何开发而成的，甚至还能为此贡献力量。其他服务和社交网络是专有的封闭系统；您永远不能确切知道它们的工作方式或者所做的事情，而且它们可能会消失不见！

无论您是需要网站生成器来创建网站还是需要简单的博客创建器，WordPress 都能提供帮助。它不仅带给您精美的设计和强大的功能，还能让您按照心中所想来自由建站。

Android 版 WordPress 支持运行 WordPress 4.0 和更高版本的自托管站点，而且所有站点均位于 WordPress.com 上。Android 版 WordPress 是开源软件，就像 WordPress 一样。详情请访问 https://apps.wordpress.com/contribute/。需要应用程序方面的帮助？请向我们发送推文 (@WPAndroid)，或者访问我们的论坛：https://android.forums.wordpress.org/forum/troubleshooting。
