WordPress for Android では、いつでもどこでもモバイルで Web 公開できます。 便利な追加機能を満載した Web サイト作成プログラムです。

作成
 - すばらしいアイデアに Web 上のホームを作りましょう。 WordPress for Android は Web サイトビルダーとブログメーカーの機能を兼ね備えたソフトウェアです。 サイト作成にご活用ください。
- 豊富な選択肢のある WordPress テーマから最適な外観を選択し、写真、色、フォントをカスタマイズできるため、非常に個性的なサイトになります。
- 内蔵のクイックスタートヒントを参考にして、基本のセットアップを済ませ、新しいサイトを成功させましょう。 (WordPress は単なるサイト作成ツールではありません。ユーザーのパートナーであり応援団でもあります) 

統計情報
 - サイトの統計情報をリアルタイムに確認してサイトのアクティビティを管理しましょう。
- 継続的に日ごと、週ごと、月ごと、年ごとの統計概要を調べ、最もトラフィックの大きい投稿やページを追跡します。
- トラフィックマップを使用して訪問者の居住国を表示します。

通知
 - コメント、「いいね !」、新規フォロワーに関する通知が届くため、サイトに対する反応をリアルタイムで把握できます。
- 新しいコメントが追加されたらすかさず返信することで、スムーズな会話の流れを維持し、読者を大切にする姿勢を示すことができます。

公開
 - エディターを使えば、更新、ストーリー、写真付きエッセーの通知など、何でも作成できます。
- お使いのカメラやアルバムから写真や動画を埋め込むと、より生き生きとした説得力のある投稿やページになります。また、アプリ内蔵のプロ写真家の無料写真コレクションからも最適の画像を選んで活用できます。
- アイデアを下書き状態で保存しておいて都合のよいときに仕上げたり、将来の新規投稿のスケジュールを設定しておいて常に新しく魅力的なサイトを作ったりできます。
- 新規読者が投稿を見つけやすくなるようにタグやカテゴリーを追加して訪問者数の増加を見守ります。

Reader
 - WordPress は単なるブログ作成ツールではありません。WordPress Reader でブロガーのコミュニティとつながるにも効果的です。 数千のタグ別トピックを調べ、新規の投稿者や組織を見つけ、興味関心の対象をフォローします。
- 「保存」機能付きで後から編集できるため、投稿にじっくり取り組めます。

共有
 - 自動共有を設定しておくと、新しい投稿を公開したとき、ソーシャルメディアのフォロワーに通知できます。 Facebook や Twitter などで自動的に相互投稿できます。
- ソーシャル共有ボタンを追加できるため、訪問者が各自のネットワークでコンテンツを共有でき、ファンをアンバサダーに格上げできます。

WordPress を利用する理由

ブログサービス、Web サイトビルダー、ソーシャルネットワークは市場にあふれています。 WordPress でサイトを作成する理由は何でしょうか ?

WordPress は Web の1/3で使われています。 趣味のブログ、大小の企業、オンラインストア、インターネット最大級のニュースサイトでも活用されています。 お好きなサイトがいくつも WordPress で稼動している可能性もあります。

WordPress では、お客様がコンテンツの所有権を留保します。  ほかのソーシャルネットワークは、利用者を財貨扱いし、利用者の投稿したコンテンツの所有権を当然のように要求しています。 その点、WordPress では、あらゆる公開コンテンツの権限が投稿者に帰属し、ほかのあらゆる場所で同時活用できます。

WordPress はオープンソース系 Web サイト制作プログラムのため、誰でもプログラムコードを閲覧でき、プログラムコードに貢献することさえ可能です。  ほかのサービスやソーシャルネットワークは、非公開のプロプライエタリシステムです。具体的な動作のしくみや機能を精確には把握できず、場合によっては市場から消え去ることもあります。

サイトを作成する Web サイトビルダーが必要な場合も、シンプルなブログメーカーが必要な場合も、WordPress は役に立ちます。 見事な設計、パワフルな機能、何でも開発できる自由があります。

WordPress for Android は、WordPress 4.0 以降が稼動するインストール型のサイトと WordPress.com 上のあらゆるサイトに対応しています。 WordPress と同様に、WordPress for Android もオープンソースです。 詳細は https://wp.me/P7jrAc-tJ でご確認ください。  アプリに関するサポートが必要ですか ? https://android.forums.wordpress.org/forum/troubleshooting にアクセスするか、@WPAndroid にツイートを送信してください。
